# ganfuzz 基于置信度的设计思路
 
- 1.先使用python实现四种变异算法
- 2.跑通一个gan模型 开源即可
- 3.跑通gan模型之后确定输入是向量 输出是图片
- 4.输入向量输入之前使用第一步的四个算法先进行变异
- 5.把输入方式改为从向量组中随机选择一个向量进行输入到gan模型中 然后让其生成图片
- 6.使用的gan模型是基于Mnist数据集训练的
- 7.再找一个Mnist的分类模型 输入是图片 输出是图片的类型 把这个模型用来预测Gan生成的图片 为每个图片生成一个标签
- 8.此时 系统可以从多个向量中选择一个向量经过变异后输出一张数字图片 然后该图片可以通过分类模型预测并分配其标签
- 9.分类模型除了给出标签还会给出置信度信息 根据置信度的值的大小来决定此时的种子向量是否应该添加到种子队列中 
    具体而言 我们先计算原本种子队列里的所有向量的平均置信度 如果此种子的置信度小于种子队列的平均置信置信度 就将其加入种子队列 并更新种子队列的置信度
- 10.第九步的过程不停循环直至500次或者队列里的平均置信度小于0.2时停止 此时我们取出队列中的最后放入的20个种子 输入到gan中将其得到的图片可视化展示

# ganfuzz 基于置信度的设计思路
 
1.生成一组初始的随机向量，作为种子队列的初始状态。
2.直接将这些随机向量输入到生成器，计算生成器的神经元覆盖率。这一步为后续的迭代提供了基准覆盖率。
3.每次迭代时，从种子队列中随机选择一个向量。
4.对选中的随机向量进行变异，生成新的向量。
5.将变异后的向量直接输入到生成器，获取生成器内部的神经元激活情况，计算变异后向量的神经元覆盖率。
6.比较变异后向量的神经元覆盖率与种子队列中向量的平均神经元覆盖率。如果覆盖率增加，则将变异后的向量加入种子队列。
7.在迭代完成后，使用rank_2方法对种子队列进行排序。这个排序的目的是优先选择那些能够最大化激活神经元的向量。
8.将这些经过筛选的高覆盖率向量输入生成器，生成并保存对应的图像。
